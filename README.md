# CivicEye AI

**Detect Early. Prioritize Intelligently. Resolve Faster. Build Smarter Cities.**

CivicEye AI is an AI-powered Public Infrastructure Intelligence Platform that enables citizens to report infrastructure issues such as potholes, broken streetlights, damaged roads, water leaks, overflowing garbage bins, and more.

The system uses Artificial Intelligence to analyze uploaded images, determine the severity of the issue, automatically assign it to the appropriate government department, and provide dashboards for citizens, department officers, and city administrators.

---

## Features

- AI-powered infrastructure issue detection
- Image upload for reporting issues
- Automatic department assignment
- Severity and priority analysis
- Citizen Dashboard
- Department Officer Dashboard
- Administrator Dashboard
- Firebase Authentication
- Firebase Firestore Database
- Firebase Storage
- Interactive Maps

## Getting Started

### Clone the repository

```bash
git clone <repository-url>
cd CivicEye-AI
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend

Open `frontend/index.html` in your browser or use a local web server.
